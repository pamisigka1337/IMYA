# IMYA — сервис аренды брендовой одежды

Веб-приложение для аренды дизайнерской одежды.
Пользователь может просматривать каталог брендовой одежды, фильтровать товары, бронировать вещи на выбранные даты и оформлять заказ.

Проект реализован как **fullstack приложение** с использованием **React + Node.js + PostgreSQL**.

---

# Основные возможности

### Каталог одежды

* просмотр всех доступных товаров
* карточки брендов
* цены аренды
* размеры

### Фильтрация

* категория
* бренд
* размер
* цена

### Карточка товара

* подробное описание
* фотографии
* бренд
* размер
* стоимость аренды
* залог

### Бронирование

* выбор даты начала аренды
* выбор даты окончания
* автоматический расчет стоимости

### Оформление заказа

* расчет итоговой суммы
* залог
* подтверждение бронирования

### Личный кабинет

* просмотр заказов
* статус бронирования

### Админ панель

* управление заказами
* просмотр бронирований

# Установка проекта

## 1. Клонировать репозиторий

```
git clone https://github.com/pamisigka1337/IMYA.git
```

```
cd IMYA
```

---

## 2. Установить зависимости

```
npm install
```

---

## 3. Настроить переменные окружения

Создать файл:

```
.env
```

Пример:

```
DATABASE_URL=postgres://postgres:password@localhost:5432/imya
PORT=5000
NODE_ENV=development
```

---

## 4. Запустить PostgreSQL

Убедитесь, что сервер PostgreSQL запущен.

---

## 5. Применить миграции базы данных

```
npm run db:push
```

или

```
npm run db:migrate
```

---

## 6. Запустить приложение

```
npm run dev
```

---

Сайт будет доступен:

```
http://localhost:5000
```
