# IMYA — сервис аренды брендовой одежды

**IMYA** — fullstack веб-приложение для аренды дизайнерской одежды.  
Пользователь может просматривать каталог брендовых вещей, фильтровать товары, выбирать даты аренды, бронировать одежду и оформлять заказ.

Проект реализован с использованием **React + Node.js + PostgreSQL**.

## Как запустить проект

### 1. Клонировать репозиторий

```bash
git clone https://github.com/pamisigka1337/IMYA.git
cd IMYA
```

### 2. Установить зависимости

```bash
npm install
```

### 3. Настроить переменные окружения

Создай файл `.env` в корне проекта и добавь:

```env
DATABASE_URL=postgres://postgres:password@localhost:5432/imya
PORT=5000
NODE_ENV=development
```

### 4. Убедиться, что запущен PostgreSQL

Перед запуском проекта база данных PostgreSQL должна быть активна.

Также нужно создать базу данных, если она еще не создана:

```sql
CREATE DATABASE imya;
```

### 5. Применить миграции

В зависимости от того, как настроен проект, выполни одну из команд:

```bash
npm run db:push
```

или

```bash
npm run db:migrate
```

### 6. Запустить проект

```bash
npm run dev
```

После запуска приложение будет доступно по адресу:

```bash
http://localhost:5000
```
