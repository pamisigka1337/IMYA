# IMYA — сервис аренды брендовой одежды

**IMYA** — fullstack веб-приложение для аренды дизайнерской одежды. Пользователь может просматривать каталог брендовых вещей, фильтровать товары, выбирать даты аренды, бронировать одежду и оформлять заказ.

Проект реализован с использованием **React + Node.js + Express + Drizzle ORM + SQLite**.

## Быстрый запуск

Больше не нужно устанавливать PostgreSQL, pgAdmin, создавать `DATABASE_URL`, выполнять `db:push` или вручную создавать таблицы.

```bash
npm install
npm run dev
```

После запуска открой ссылку из терминала, обычно:

```bash
http://localhost:5000
```

При первом запуске проект автоматически:

- создаст локальную SQLite-базу в файле `data/app.db`;
- создаст все таблицы;
- добавит тестовые товары, пункт выдачи и пользователей для входа.

## API endpoints

| Метод | Endpoint | Описание |
| --- | --- | --- |
| GET | `/api/items` | Получить каталог активных товаров |
| GET | `/api/items/:id` | Получить информацию о конкретном товаре |
| POST | `/api/auth/register` | Зарегистрироваться |
| POST | `/api/auth/login` | Войти |
| POST | `/api/bookings` | Создать бронирование |
| GET | `/api/bookings/my` | Получить бронирования текущего пользователя |
| GET | `/api/admin/items` | Получить все товары для админа |
| POST | `/api/admin/items` | Создать товар для админа |
| PATCH | `/api/admin/items/:id` | Изменить товар для админа |
| GET | `/api/admin/bookings` | Получить все бронирования для админа |

## Требования для запуска

- **Node.js** 22+ или 24+
- **npm** 9+

PostgreSQL и pgAdmin не нужны.

## Сброс локальной базы

Если нужно начать с чистой базы, останови сервер и удали файл:

```bash
rm data/app.db
```

На Windows PowerShell:

```powershell
Remove-Item data/app.db
```

При следующем `npm run dev` база, таблицы и тестовые данные создадутся заново.
